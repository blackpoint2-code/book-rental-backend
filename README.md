📚 도서 대여 서비스 (프로젝트명)
"누구나 쉽게 도서를 대여하고 관리하는 커뮤니티형 도서 서비스"

📅 프로젝트 개요
소개: 도서 검색, 대여/반납 관리, 회원 간의 도서 정보 공유 및 소통이 가능한 웹사이트
개발 기간: 2026.07.08 ~ 진행 중

🛠 기술 스택

카테고리	기술

운영체제	Windows 11

언어	Java, JavaScript

프론트엔드	React, CSS

백엔드	Spring Boot

데이터베이스	PostgreSQL

ORM	JPA (Spring Data JPA)

빌드 도구	Gradle

개발 도구	IntelliJ IDEA, VS Code

🚀 주요 기능
도서 검색 및 조회: 제목, 저자별 도서 검색 및 상세 정보 확인
도서 대여/반납: 실시간 대여 상태 확인 및 도서 신청 기능
사용자 마이페이지: 나의 대여 기록 및 연체 관리
관리자 기능: 신규 도서 등록, 회원 관리 및 대여 현황 파악

💡 프로젝트 성과 및 기대 효과
기술 역량: JPA를 활용한 객체 지향적 DB 설계 및 운영 경험
협업/관리: Git을 활용한 버전 관리 및 체계적인 이슈 해결
프론트엔드: React를 통한 SPA(Single Page Application) 개발 능력 확보

⚠️ 이슈 발생 및 해결 (예시)
상황: 프론트엔드-백엔드 간 데이터 통신 시 CORS 오류 발생
원인: 다른 포트(3000 vs 8080) 간의 요청 차단
해결: Spring Boot의 @CrossOrigin 설정 및 프록시 설정을 통한 통신 허용
